## 1.项目介绍
1. 由于本项目主要是巩固网络编程，IO流和多线程的知识，所以并没有加入数据库在管理账号，望谅解。
2. 当用户A申请与B聊天时，经过B同意之后，服务器将建立A与B之间的聊天通道，此时A和B将不再展示主页面，直到一方输入“exit”退出私聊。
3. 当用户A向用户B发送文件时，为保护用户B的安全，在服务器确认用户B同意接受该文件时，才会将文件传输给用户B。
